## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/username/yebab.git
   cd yebab
   ```

2. **Open in browser**
   ```bash
   start invitation-management.html  # On Windows
   open invitation-management.html   # On macOS
   xdg-open invitation-management.html  # On Linux
   ```
